Simulation of a multi compartment model (passive neurite)¶

Regarding to Kirchhoff's Current Law it holds for comparment $j$ that: $ \frac{I_e}{A} = C_m \frac{dV}{dt} + \frac{E_L}{R_m}$

The voltage for a compartment $j$ with an input current $I_e$ is therfore given by the DEQ: $C_m \frac{dV}{dt} = -\frac{E_L}{R_m} + \frac{I_e}{A}$

In a multi-compartment model compartment $j$ is connected by an axial resistance $R_a$ to adjoining compartments $j-1$ and $j+1$ with voltages $V_{j-1}$ and $V_{j+1}$. This leads to inflowing currents $\frac{(V_{j-1}-V_j)}{R_a}$ and $\frac{(V_{j+1}-V_j)}{R_a}$.

In order to fullfill the Kirchhoff's Current Law in this case it must hold that: $$\frac{I_e}{A} + \frac{(V_{j-1}-V_j)}{R_a} + \frac{(V_{j+1}-V_j)}{R_a} = C_m \frac{dV}{dt} + \frac{E_L}{R_m}$$ Therefore the voltage in compartment $j$ is given by the DEQ: $$C_m \frac{dV}{dt} = -\frac{E_L}{R_m} + \frac{I_e}{A} + \frac{(V_{j-1}-V_j)}{R_a} + \frac{(V_{j+1}-V_j)}{R_a}$$(in the case that $j$ is an terminal compartment one of these inflowing currents drops)

def multi_compartment_simulation(N, C_m, R_m, R_a, E_m,input_function, delta_t, t):
    steps = np.arange(0,t,delta_t) 
    compartment_potential = np.zeros((N,steps.shape[0])) # initialize with zero
    
    
    
    for j, t in enumerate(steps[:-1]):
        for i in range(1,N-1):  # stop before killed-end compartment
                post  = (compartment_potential[i+1,j] - compartment_potential[i][j])/R_a
                pre = (compartment_potential[i-1,j] - compartment_potential[i][j])/R_a
                current = input_function(i,t) + (E_m - compartment_potential[i][j])/R_m 
                deltaV_detlat = (pre +current + post)/C_m 
                compartment_potential[i,j+1] = compartment_potential[i,j] + delta_t * deltaV_detlat
                
        compartment_potential[0,j] = compartment_potential[1,j] # Boundary condition for sealed terminal compartment 
    return compartment_potential, steps

#------------------------------------------Electric Properties---------------------------------------------------------

C_m = 62.8 * 10**-12 # F
R_m = 1.59 * 10**9 # Ohm
R_a = 0.0318 * 10 **9 # Ohm
E_l = E_m = 0 # V

#--------------------------------------------Time Properties-----------------------------------------------------------
delta_t = 0.0001 # seconds 
j_e = 20 # Compartment index 
t_e = 0.02 # seconds
I_0 = 10*10**-12 # A
t_s = 0.4 # seconds


#------------------------------------------Current Functions-----------------------------------------------------------
def step_current(j,t):
    if(t<t_e or j != j_e):
        return 0
    if(t_e <= t and j==j_e):
        return I_0
    
def rectangular_current(j,t):
    if(t< t_e or t_s <= t or j != j_e):
        return 0
    if(t_e <= t and t < t_s and j == j_e):
        return 10*I_0

Discussion¶

The functional form of the spatially dependent stationary solution shows a two-sided exponential decay with equal slope. The current injected in compartment 20 leads to an increase in voltage in this compartment. From here it equally expends to adjoining compartments in both directions. Looking at the voltage change over time you can see that all compartments follow the same functional form with different amplitudes.
